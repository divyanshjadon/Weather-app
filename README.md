Weather App

A simple weather app built with vanilla JavaScript and the OpenWeatherMap API.

Features

Search weather by city name
Displays temperature, feels like, humidity, and wind speed
Handles invalid city names with a friendly error message

Tech Stack

HTML, CSS, JavaScript
OpenWeatherMap API

How to run

Clone the repo
Open index.html in your browser
Enter a city name and click Search

What I learned

Fetching data from a public API using JavaScript fetch()
Handling API responses and errors
Dynamically updating the DOM with real data
